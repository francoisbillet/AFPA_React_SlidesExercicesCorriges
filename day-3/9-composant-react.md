Un composant, c'est juste une fonction :

```jsx
function MonComposant() {
  return (
    <div>
      <b>composant</b>
    </div>
  );
}
```

```jsx
const MonComposant2 = () => {
  return (
    <div>
      <pre>composant</pre>
    </div>
  );
};
```

```jsx
const MonComposant3 = () => {
  return "Un composant hyper simple";
};
```
